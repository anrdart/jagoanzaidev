# Jagoan Zaidev

Website edukasi untuk belajar server dari nol, dengan bahasa yang santai dan mudah dipahami.

## Tech Stack

- **Astro** - Framework untuk static generation yang cepat
- **React** - UI Library untuk komponen interaktif
- **Tailwind CSS** - Untuk styling modern dan responsif
- **Lucide React** - Icon library yang bersih dan modern

## Fitur

- Hero section dengan CTA yang menarik
- Penjelasan tentang server dengan analogi sederhana
- 3 Aturan Emas (Domain, Hosting, VPS) dengan layout timeline
- Footer dengan social links
- Desain pastel modern tanpa gradient
- Responsif untuk semua ukuran layar

## Instalasi

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build

# Preview production build
npm run preview
```

## Desain

Website ini menggunakan:
- Warna pastel (slate, sage, blue-gray, coral, mint)
- Flat colors tanpa gradient
- Font Plus Jakarta Sans untuk tampilan modern
- Hover effects halus pada kartu dan tombol
- Bahasa Indonesia yang santai dan friendly
